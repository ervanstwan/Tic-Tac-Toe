
# 🎯 Tic-Tac-Toe Game CI/CD Pipeline

A simple Tic-Tac-Toe game with a full GitLab CI/CD pipeline.

## 🚀 Tech Stack
- Node.js
- GitLab CI/CD

## ⚙️ Pipeline Stages
```
| Stage       | Description                |
|------------------------------------------|
| 🛠 Build    | Install dependencies       |
| ✅ Test     | Run dummy tests           |
| 🔒 Security | Run `npm audit`           |
| 🚀 Deploy   | Simulate deployment       |
```

## ▶️ Run Locally
```
npm install
npm start
```

Enjoy the game!
